DeepLearning SDK for Geospatial Data

Welcome to DeepCore

DeepCore is a machine learning abstraction framework and utility toolkit, written in C++ by DigitalGlobe. It allows a user to download, perform object detection, and manipulate geospatial vector files. DeepCore is intended to be a machine learning framework agnostic toolkit, allowing for a simple, clean, consistent programmatic interface. As new machine learning techniques and frameworks emerge, they can easily be integrated into DeepCore. This allows developers using DeepCore to easily extend their applications with the latest technology, without having to worry about the complexities of each framework or technique.

DeepCore's machine learning features can also be accelerated by the use of Nvidia Graphics Processing Units (GPUs) using CUDA technology. By enabling GPU mode, the process of object detection becomes very quick, allowing for faster and more efficient processing of large areas. The use of GPUs to accelerate machine learning and object detection processes is highly recommended.

DeepCore is split into libraries that are grouped by function. The core libraries are:

  • Utility - contains toolkit type functions that are very general and don’t fit into any of the other categories. This includes logging via Google glog, zip file reading and writing, basic math, and other basic functions.
  • Network - contains functions and classes for communication across networks, including http and json-rpc requests
  • Vector - contains functions and classes for writing vector features out to disk or network sources. Supported formats are:
    • Shapefile
    • KML
    • PostGRES/PostGIS
    • Elasticsearch
    • GeoJSON
  • Imagery - contains functions and classes for reading imagery from a variety of sources. Using GDAL and OpenCV, most image formats can be read in for processing.
  • Classification - contains the functions and classes necessary to do classification and object detection with a supported model. Currently, only Caffe models are supported, but additional frameworks such as Microsoft’s CNTK are coming online soon.

Sample Application - gbdxm

The GBDXM packaging tool is a DeepCore based executable that enables the user to package a machine learning model for use with other DeepCore based tools. It takes in a supported model format as an input, and packages the model files into a single easy-to-use, easy-to-share container. The DeepCore framework can then use that model package to perform machine learning and object detection tasks. Currently, only Caffe models are supported, but other model formats are forthcoming as additional machine learning frameworks are incorporated into DeepCore. The GBDXM application can be downloaded via link or by using the button on the right.

Sample Application - OpenSpaceNet

OpenSpaceNet is a DeepCore based sample application that uses DigitalGlobe satellite imagery as an input source for the object detection. Using DigitalGlobe image services, such as the DigitalGlobe Maps API or the My DigitalGlobe imagery service, the user can access current satellite imagery for use in machine learning. OpenSpaceNet allows the user to select an area for detection, specify a model to use during detection, and get the results back in a geospatial format suitable for use in a GIS application or service. The OpenSpaceNet application can be downloaded via link or by clicking the button on the right.

Detailed Description

Represents an abstract feature detector implemented as a sliding window classifier. Implements Detector and Classifier interfaces as special cases for a sliding window detector.

const cv::Size & dg::deepcore::classification::SlidingWindowDetector::overrideSize ( ) const

Gets the override window size.

If override window size is specified, the sliding windows will appear to be of the override size. The image slice passed to the classifier will have the image slice of the override size centered in it with the border filled in by Gaussian noise.

Returns
The override window size.

\begin{DoxyReturn}{Returns}

\end{DoxyReturn}
Verbose logging for C\+U\+RL connection info and headers is sent to the logger with level T\+R\+A\+CE and channel \hyperlink{classdg_1_1deepcore_1_1network_1_1_http_downloader}{Http\+Downloader}. It includes the pointer for the connection handle.

Note that verbose logging does not setup a sink for the logs, just the source. A sink is still required to view the messages.

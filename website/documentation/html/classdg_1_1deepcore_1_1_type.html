Detailed Description

Represents a data type.

Type class provides a way to describe data type in a portable, runtime reflective way.

To get an instance of the Type object for a data type use Type::find or Type::get.

staticnoexcept

Finds a Type with the given name from the type registry.

Parameters
nameThe Type name.
Returns
A pointer to the Type, nullptr is returned if the type is not found. The pointer is owned by the type registry and should not be freed.

Example

auto t = Type::get("Int32");
if(!t) {
std::cerr << "Type Int32 not found" << std::endl;
}

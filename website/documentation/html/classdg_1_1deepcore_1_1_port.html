Detailed Description

Base class for Node's ports.

Note
You should not inherit from this class directly, use PortT instead.

Example Port Declaration

namespace port {
// Declare the port.
using Foo = PortT<Int32, String>;
namespace type {
// Register the port type
const TypeT<Foo> Foo("Foo");
}
}

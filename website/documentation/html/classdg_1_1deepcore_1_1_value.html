Detailed Description

A class whose instance can hold instances of any type that has value semantics.

The type is said to have value semantics when it satisfies the DefaultConstructible, CopyConstructible, MoveConstructible, CopyAssignable, and MoveAssignable concepts.

Example

Value v1 = (int32_t)42;
assert(*v.as<int32_t> == 42);
assert(v.as<int16_t> == nullptr);
assert(v.cast<int16_t>() == 42);
v2 = v1.cast(types::Int16);
assert(*v.as<int16_t> == 42);
auto v3 = v2.moveCast(types::Float32);
assert(v3.as<float>() == 42.0)
assert(v2.type() == types::Void);

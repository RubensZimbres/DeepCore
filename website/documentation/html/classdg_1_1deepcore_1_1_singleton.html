dg::deepcore::Singleton< T, S > Class Template Reference
Utility

Detailed Description

template<class T, class S = Storage::GLOBAL>
class dg::deepcore::Singleton< T, S >

A thread-safe singleton class holder.

Template Parameters
TClass from which to create a singleton.
SStorage specifier, default is Storage::GLOBAL.

Examples

Using Singleton on an Existing Class

Class declaration:

class MyClass
{
public:
void doSomething();
}

Singleton declaration:

using MySingleton = Singleton<MyClass>;

Calling a singleton method:

MySingleton::instance().myMethod();

Using Singleton as a Base Class For a Singleton-only Class

Declaring the singleton class:

class MySingleton final : public Singleton<MySingleton>
{
public:
void doSomething();
private:
friend class Singleton;
MySingleton();
};

Calling a singleton method:

MySingleton::instance().myMethod();

Creating and using a thread-local singleton:

using MySingleton = Singleton<MyClass, SingletonStorage::THREAD_LOCAL>;
// ...............................................
MySingleton::instance().myMethod();

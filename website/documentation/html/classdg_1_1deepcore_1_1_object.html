dg::deepcore::Object Class Reference
Processing » Type System

Detailed Description

The base class for all the processing pipeline objects.

The Object class provides a common base for all classes. It provides reference-counted memory management through Object::ptr() and Object::wptr(), runtime-reflectable typing through the use of Type, and a runtime factory which allows creation of Object descendants by name or Type.

Do not inherit from Object directly, use ObjectT instead.

#include <process/Object.h>

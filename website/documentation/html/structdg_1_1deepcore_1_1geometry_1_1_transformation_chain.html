Example

An example of usage of this is creating a lat/lon-to-pixel transformation for a GeoImage. image in this example is an instance of GeoImage:

TransformationChain llToPixel {
image_->spatialReference().fromLatLon(),
image_->pixelToProj().inverse()
};

bool dg::deepcore::geometry::TransformationChain::operator== ( const Transformation &  rhs) const
overridevirtual

Returns whether rhs is equivalent to this.

Parameters
[in]rhsThe other transformation.
Returns
true if rhs is equivalent to this.
Note
This function will compare to chains of transformations as is, without making sure they can be made equivalent. To make sure the comparison is correct, call TransformationChain::compact() on this transformation. If rhs is also a TransformationChain, call TransformationChain::compact() on rhs as well.

bool dg::deepcore::network::HttpDownloader::verboseLogging ( )
static

Is verbose logging of CURL connections enabled?

Returns
true if verbose logging be turned on.

Verbose logging for CURL connection info and headers is sent to the logger with level TRACE and channel HttpDownloader. It includes the pointer for the connection handle.

Note that verbose logging does not setup a sink for the logs, just the source. A sink is still required to view the messages.

Detailed Description

template<class I, class O>
struct dg::deepcore::PortT< I, O >

A helper template to declare Port types.

The template implements the inputType() and outputType() virtual methods using the template arguments to construct the return Types.

Template Parameters
IInput data type.
OOutput data type.

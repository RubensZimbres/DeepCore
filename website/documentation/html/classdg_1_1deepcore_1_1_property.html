dg::deepcore::Property Class Referenceabstract
Processing » Framework

Detailed Description

Base class for Node's properties.

Note
You should not inherit from this class directly, use PropertyT instead.

Example Property Declaration

struct MyType { ... };
namespace prop {
// Declare
using MyType = PropertyT<MyType>;
}
// .................
Type::add<MyType>("MyType");

#include <process/Property.h>

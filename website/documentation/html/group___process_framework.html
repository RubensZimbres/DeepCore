enum dg::deepcore::NodeState
strong

Represents a state of a Node.

Enumerator
STOPPED 

The Node is stopped. This is the initial and final state of a node.

IDLE 

The Node is running, but currently has no work to do. The node is waiting on data to appear on one of its ports or to be notified to wake up.

RUNNING 

The Node is currently processing data.

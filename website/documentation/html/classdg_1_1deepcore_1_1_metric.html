Detailed Description

Base class for Node's metrics.

Note
You should not inherit from this directly, use MetricT instead.

Example Metric Declaration

struct MyType { ... };
namespace met {
// Declare
using MyType = MetricT<MyType>;
}
// .................
Type::add<MyType>("MyType");

#include <process/Metric.h>

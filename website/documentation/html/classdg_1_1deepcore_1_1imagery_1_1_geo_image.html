virtual void dg::deepcore::imagery::GeoImage::readBlocks ( const std::vector< cv::Point > &  blocks)
pure virtual

Queues blocks to be read. The blocks are then passed back using readFunc().

Parameters
[in]blocksThe block to read.
Precondition
Read function must be set using setReadFunc() prior to calling this method.

This function can either queue blocks for reading or read them synchronously, whatever is more efficient for the implementation.

Implemented in dg::deepcore::imagery::MapServiceImage, and dg::deepcore::imagery::GdalImage.

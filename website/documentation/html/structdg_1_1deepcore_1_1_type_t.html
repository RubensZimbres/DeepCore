Detailed Description

template<class T>
struct dg::deepcore::TypeT< T >

Represents a data type equivalent to T.

Template Parameters
TThe underlying data type.

This helper template class is used to instantiate Type objects and add them to the type registry. Any data type that participates in the processing framework as a value has to create an instance of this class, usually as a static const value.

In order to access the type information after the initial declaration, use Type::get.

Example

struct MyType
{
int value;
}
namespace types {
static const TypeT<MyType> MyType("MyType");
}
// ....
// Later in the code
auto& t1 = Type::get<MyType>();
// or
auto& t2 = Type::get("MyType");
// or
auto& t2 = Type::get(typeid(MyType));
assert(t1 == types::MyType)
